# README

## アプリ概要

これはお金の管理が苦手な人が管理できるようになる事を目的として作成されたアプリである。		

## 開発言語

* Ruby 2.6.5
* Ruby on Rails 5.2.6

## 就業Term内で取り入れる技術

* お気に入り機能
* Ajaxを使ったコメント機能

## カリキュラム外の技術
* chartkick（グラフ作成機能）

## その他の技術(時間に余裕があれば実装)
* kaminari
* ransack
* active_hash

## 実行手順

```bash
$ git clone git@github.com:hiroaki-kudo/money_management.git
$ cd money_management
$ bundle install  
$ rails db:create db:migrate  
$ rails s
```
